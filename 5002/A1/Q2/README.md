# FP-tree ![CI status](https://img.shields.io/badge/build-passing-brightgreen.svg)
This is a FP tree toy demo.

## Usage

```
$ python FPTree.py -p path_to_csv -s support
```

or,

```python
from FPTree import *
fpt=FPTree()
fpt.fit(transactions, support)
```
